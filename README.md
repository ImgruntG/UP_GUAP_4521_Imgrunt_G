🎮 Paper.io Multiplayer (Node.js + Docker)

Многопользовательская браузерная игра по мотивам Paper.io, реализованная с использованием Node.js, Socket.IO, Canvas и упакованная в Docker-контейнер для простого развертывания.

📌 Описание

Проект представляет собой клиент-серверную онлайн-игру, в которой игроки соревнуются за захват территории на общей карте в режиме реального времени.

Особенности проекта:

- многопользовательский режим

- синхронизация в реальном времени

- система захвата территории

- игровой бот

- динамическая камера и визуальные эффекты

- Docker-развёртывание в один шаг


🚀 Быстрый запуск (Docker)

1. Скачать образ

Скачайте Docker-образ по ссылке:

https://drive.google.com/file/d/1B8hR5xvWrcM2zmYCxjk4zitQCjEpAkQN/view?usp=drive_link

2. Подготовка проекта

Поместите в одну директорию следующие файлы:

/project-folder

 ├── paperio-image.tar
 
 ├── install.sh

3. Выдать права на запуск

chmod +x install.sh

4. Запуск

./install.sh


⚙️ Требования

Перед запуском убедитесь, что система поддерживает:

- Linux (Ubuntu/Debian рекомендуется)

- Docker (установится автоматически при отсутствии)

- curl

- доступ к порту 3000

🧠 Как работает install.sh

Скрипт выполняет автоматическое развертывание:

- Проверяет наличие Docker

- Устанавливает Docker при необходимости

- Загружает образ paperio-image.tar

- Удаляет старый контейнер game

- Создаёт новый контейнер

- Пробрасывает порт 3000

- Запускает сервер (node server.js)

- Проверяет доступность сервера

- Выводит IP-адрес для подключения

🌐 Доступ к игре

После успешного запуска игра будет доступна по адресу:

http://<SERVER_IP>:3000

📁 Структура контейнера

Внутри Docker-контейнера проект расположен в:

/var/www/paperio

🧪 Проверка работы

Если сервер не запускается:

docker logs game

🛑 Возможные проблемы


❌ Контейнер не запускается
Проверьте имя файла: paperio-image.tar
Проверьте права доступа
Убедитесь, что Docker запущен


❌ Игра недоступна
Проверьте порт 3000
Проверьте firewall / iptables
